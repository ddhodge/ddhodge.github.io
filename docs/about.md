This site is just me testing Jekyll + Slate theme.

Some front matter you might use:

- `layout: default` ← almost always needed with remote themes
- `title:` ← appears in <title> and often as <h1>
- `permalink:` ← optional, controls exact URL (good to be explicit)

You can add more variables if the theme supports them.
